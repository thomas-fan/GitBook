# 栈和队列

## 栈,队列, 双端队列

### 特点

+ 栈先入后出

+ 队列先入先出

+ 添加,删除为 O(1)

+ 查询为 O(n)

+ 插入 O(1)
+ 取出O(logN)
+ 底层具体实现的数据结果较为多样和复杂: heap, bst, rb-tree, treap
